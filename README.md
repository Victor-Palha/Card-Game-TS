# Card Game
 web application using web sockets to create a multiplayer card game
 * The application use MySQL database and MongoDB
 * All the application are make to running on web
## Technologies
* Node.JS
* Express
* MySQL
* MongoDB
* React.js
* TypeScript
* CSS3
